Project Architecture

Je souhaite donc annoncer avec la plus grande fierté, que j'active dans l'immédiat le site de pack openning du cégep de Sorel-Tracy, plus communément connu sous le nom de : POTI. (Ps: Si vous avez un meilleur nom, n'hésitez pas !)

Le site va contenir les features suivantes :

  • Ouverture de pack à tous les 6H (Maximum: 2 Pack)
  • Système utilisateur en Standalone avec le email du cégep
  • Boutique en ligne avec système monétaire regroupé
  • Collections de cartes:
    • Les cours techniques (81 cartes)
    • Les profs (21 cartes)
    • Les locaux (18 cartes, le local des profs est le plus rare)
  • Système de gambling avec des doublons
  • Monnaie virtuelle appelée POTI
  • Inventaire des cartes
  • Projet
    • Server
      • Spring-Boot
        • repository
        • model
        • controller
        • services
        • config
        • CLI tools
        • utilities
        • test
    • Frontend
      • React + TS
        • components
        • views
        • types
        • api
        • hooks
        • contexts
        • state management
        • styles (CSS, SCSS)
        • test (unit and integration)
      • UI Frameworks (Bootstrap, Material-UI)
    • Base de données
      • BD public pour les cartes et autres pacotilles
        • init.sql (initialization script)
        • mock_data.sql (mock data script)
      • BD privée pour les utilisateurs
        • init.sql (initialization script)
        • mock_data.sql (mock data script)
    • Docker-compose
      • Dockerfile pour API
      • Dockerfile pour les BD's
      • docker-compose.override.yml
      • volume management

Pour ceux qui sont intéressés, je vous invite à en parler avec d'autres programmeurs et à dire dans ce chat si cela vous intéresse, ainsi que sur quelle partie vous aimeriez travailler !

Les rôles sont les suivants :

  • Backend
  • Frontend
  • Base de données

Pour tous ceux qui participeront, nous allons vous faire une carte spéciale à votre effigie, dans une 4ème collection.

Sinon, c'est surtout pour apprendre et s'amuser tous ensemble !
